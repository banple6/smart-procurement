# 三公鲜配公测快速使用说明

适用对象：管理员、子单位用户、现场测试人员

## 一、安装 App

1. 打开 Web 登录页。
2. 点击“下载 Android App”。
3. 下载完成后安装。
4. 如手机提示“禁止安装未知来源应用”，请按手机提示允许本次安装。
5. 安装后桌面应用名称应显示为“三公鲜配”。

## 二、管理员怎么用

1. 打开三公鲜配 App。
2. 使用管理员账号登录。
3. 进入“我的 → 帮助与教程”查看管理员教程。
4. 先检查：
   - 子单位是否完整；
   - 默认配送点是否正确；
   - 食材、价格、库存是否正确；
   - 订单状态是否能从接单流转到完成。
5. Web 端可使用 App 扫码登录管理后台。

管理员日常路径：

```text
工作台 → 订单管理 → 接单 → 开始备货 → 确认发货 → 完成订单
```

## 三、子单位怎么用

1. 打开三公鲜配 App。
2. 使用子单位账号登录。
3. 首页选择食材。
4. 进入采购清单核对数量、金额和配送点。
5. 提交订单。
6. 在订单页查看配送进度。
7. 收到货后点击确认收货。

子单位日常路径：

```text
首页 → 加入清单 → 清单 → 提交订单 → 订单 → 确认收货
```

## 四、网页登录

1. 打开 Web 登录页。
2. 使用已登录的三公鲜配 App 扫描二维码。
3. App 上确认登录。
4. 系统会按角色自动进入：
   - 管理员：管理后台；
   - 子单位：单位网页版。

## 五、遇到问题怎么办

- 忘记密码：联系管理员重置密码。
- 账号停用：联系管理员检查账号状态。
- 所属单位停用：联系管理员检查单位状态。
- 库存不足：减少数量或联系管理员补库存。
- App 无法下载：使用 Web 登录页的“复制下载链接”发给现场人员排查。
- 页面异常显示 `undefined` 或空白：截图并记录操作路径、账号、时间。

## 六、公测反馈记录建议

反馈时请提供：

- 账号；
- 所属单位；
- 手机型号；
- 操作时间；
- 操作步骤；
- 截图或录屏；
- 是否能重复出现。
