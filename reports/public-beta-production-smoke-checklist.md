# 公测生产冒烟检查清单

执行时间：待生产部署后填写

## 部署前

- [ ] 创建或确认 ECS 快照。
- [ ] 备份生产 SQLite 数据库。
- [ ] 记录生产数据库 SHA-256。
- [ ] 记录当前 Docker 镜像、容器名和启动参数。
- [ ] 记录当前 APK production 发布版本。
- [ ] 确认维护窗口。
- [ ] 确认无正式用户正在操作。

## 服务健康

- [ ] `GET /api/v1/health` 返回 `{"status":"ok"}`。
- [ ] `GET /api/v1/health/ready` 返回 `{"status":"ready"}`。
- [ ] 正式容器无重启。
- [ ] 正式容器无 OOM。
- [ ] Nginx 无持续 5xx。
- [ ] SQLite `integrity_check` 为 `ok`。
- [ ] SQLite `quick_check` 为 `ok`。

## Web 未登录

- [ ] `/login` 可打开。
- [ ] 登录页显示“三公鲜配”。
- [ ] 登录页显示 App 下载入口。
- [ ] `/download` 可打开。
- [ ] `/help` 可打开。
- [ ] `/help/admin` 可打开。
- [ ] `/help/unit` 可打开。
- [ ] 页面不出现 `undefined`、`null`、旧品牌名称或本机路径。

## Web 管理员

- [ ] 管理员扫码登录进入 `/admin/dashboard`。
- [ ] 管理员无法因刷新丢失会话。
- [ ] 管理员后台显示“下载 App”和“帮助”入口。
- [ ] 订单列表可加载。
- [ ] 食材列表可加载。
- [ ] 单位列表可加载。
- [ ] 点击下一步订单操作有反馈，不重复提交。
- [ ] 退出登录后无法继续访问后台。

## Web 子单位

- [ ] 子单位扫码登录进入 `/unit/home`。
- [ ] 子单位无法访问 `/admin/dashboard`。
- [ ] 子单位只能看到本单位订单。
- [ ] 子单位首页显示下载 App 入口。
- [ ] 子单位帮助页面可打开。

## App

- [ ] 真机安装 production APK。
- [ ] App 桌面名称为“三公鲜配”。
- [ ] 管理员登录成功。
- [ ] 子单位登录成功。
- [ ] App 更新检查可用。
- [ ] 帮助与教程可打开。
- [ ] 扫码登录 Web 可用。

## 小范围业务链路

- [ ] 管理员确认有正式单位和默认配送点。
- [ ] 管理员确认有可申领食材。
- [ ] 子单位加入清单。
- [ ] 子单位提交一笔公测测试订单。
- [ ] 管理员接单。
- [ ] 管理员开始备货。
- [ ] 管理员确认发货。
- [ ] 子单位确认收货。
- [ ] 台账可查看。
- [ ] Excel 可导出。

## 结束确认

- [ ] 公测测试订单已明确标记，不混入正式统计口径。
- [ ] 无 HTTP 5xx 持续出现。
- [ ] 无 `database is locked`。
- [ ] 无负库存。
- [ ] 无跨单位数据可见。
- [ ] 回滚材料仍完整可用。
